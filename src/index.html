Binary Tree Visualization

Tree Input

Tree input is a string that represents the structure of the binary tree. It consists of space-separated 32 bit signed integer values and the alphabet 'n' (case-insensitive) signifying a NULL node.


The values are essentially the result of level order traversal of the resulting binary tree. To signify a null node on a level, the alphabet 'n' (case-insensitive) is used. Null nodes have no children so they are skipped when rendering the next level (children of the current level) of the binary tree.


Some Examples:


1 2 3

1 N 3

1 N 3 4 5
